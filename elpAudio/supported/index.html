Supported file formats:

  • Audio Interchange File Format (.aiff)
  • Advanced Systems Format (.asf;.asx)
  • DLS file format (.dls)
  • Free Lossless Audio Codec (.flac)
  • FMOD sound banks (.fsb)
  • FT2/IT2/ST3/PT2 modules (.mod;.xm;.s3m;.it;)
  • MIDI format (.mid)
  • MPEG Layer 2/3 (.mp2;.mp3)
  • Playlists (.m3u, .ram, .axf, .wax, .wvx, .wpl, .w3c, .b4s, .p2p, .kpl, .itl, .rdf, .pls, .elf and .epl (last two are elpAudio's own playlist formats))
  • Windows Media Audio (.wma)
